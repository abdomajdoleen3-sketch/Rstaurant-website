# Restaurant Website

A responsive website for a restaurant, showcasing our menu, about us section, and contact information.

## Features
- **Home:** Landing page for the restaurant.
- **About:** Information about our restaurant.
- **Menu:** Includes a variety of delicious items:
    - **Pizza:** Vegetarian, Pepperoni, and Margherita.
    - **Burgers:** Cheese Burger, Chicken Burger, and Classic Burger.
    - **Drinks:** Laban, Pepsi, Orange Juice, Carrot Juice, and Lemon Juice.
- **Contact:** Displays our location, phone number, and operating hours.

## Technologies Used
- HTML
- CSS
